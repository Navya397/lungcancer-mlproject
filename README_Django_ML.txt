# 🧠 Lung Cancer Prediction Web App (Django + ML)

This is a Django web app integrated with a machine learning model to predict the risk of lung cancer based on user input.

## 🚀 How to Run

1. Clone the repo or download the files
2. Navigate into the project folder:
   ```bash
   cd accounts
   ```
3. Run the server:
   ```bash
   python manage.py runserver
   ```

## 📂 Project Structure

- `manage.py` – Django entry point
- `accounts/` – Django project folder
- `ml/` – Machine learning app
- `templates/` – HTML templates
- `model.pkl` – Trained ML model
- `columns.pkl`, `scalar.pkl` – Preprocessing files

## ⚙️ Tech Stack

- Python
- Django
- scikit-learn
- pandas
